# Palindrome Checker App

**Version:** 1.0  
**Author:** aadhithya

## 📌 Description
Palindrome Checker App is a simple Java console application.  
Currently, it displays a welcome message for the Palindrome Checker App Management System.

This project is created as a basic Java application structure example.

---

## 🚀 Getting Started

### ✅ Prerequisites
- Java JDK 8 or higher
- Command Prompt / Terminal

### ▶️ How to Run

1. Clone the repository or download the source code.

2. Open terminal/command prompt in the project folder.

3. Compile the program:
   ```bash
   javac PalindromeCheckerApp.java
   ```

4. Run the program:
   ```bash
   java PalindromeCheckerApp
   ```
